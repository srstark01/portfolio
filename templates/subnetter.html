Subnet Calculator

Here is a simple subnet calculator I created from scratch using python. The subnet calculator uses a method called ANDing. ANDing calcualtes a network address by converting a given IP adress and subnet mask to their binary equilivents. Once this is done we can calculate things like range, from first adress to last address, broadcast address and the total number of available hosts. If you'd like to learn more about how ANDing works check out this site.
